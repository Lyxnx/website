Simple Hangman Game

This was a little side project I decided to undertake. It is a game of hangman played entirely on a command line interface (CLI).

User inputs:

  • -w --word WORD - This would set the game to use the specified word instead of a randomly chosen one.
  • -s --source SOURCE_FILE (default current_dir/words.txt - This would allow the user to set a source file of words for the program to randomly choose from.

The source file could be any file as long as it's a text file with each entry being separated by a new line character (\n).
